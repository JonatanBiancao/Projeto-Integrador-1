Conceitos Técnicos

Servidor FTP

Um servidor FTP (File Transfer Protocol) é um software que permite a transferência de arquivos entre computadores em uma rede, como a Internet. Ele opera no modelo cliente-servidor, onde:

  • O cliente FTP inicia uma conexão com o servidor
  • A autenticação é realizada através de nome de usuário e senha
  • O usuário pode então navegar, baixar e enviar arquivos para o servidor

Os servidores FTP utilizam as portas 20 e 21 por padrão: a porta 21 para comandos de controle e a porta 20 para transferência de dados. Eles são amplamente utilizados para hospedagem de sites, compartilhamento de arquivos e backups.

JavaScript

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos. Criada inicialmente para tornar as páginas web interativas, hoje o JavaScript é usado tanto no front-end quanto no back-end do desenvolvimento web.

Características principais:

  • Permite manipulação dinâmica do DOM (Document Object Model)
  • Possibilita a criação de elementos interativos como sliders, formulários dinâmicos e animações
  • Suporta programação assíncrona, facilitando operações como requisições AJAX
  • Com Node.js, pode ser usado para desenvolvimento de servidores e aplicativos desktop

JavaScript é essencial no desenvolvimento web moderno e forma, junto com HTML e CSS, a tríade fundamental das tecnologias web.

Docker

Docker é uma plataforma aberta que permite desenvolver, enviar e executar aplicações em contêineres. Contêineres são unidades padronizadas de software que empacotam código e todas as suas dependências para que o aplicativo seja executado de forma rápida e confiável de um ambiente de computação para outro.

Vantagens do Docker:

  • Isolamento: aplicações executam em ambientes isolados, evitando conflitos
  • Portabilidade: contêineres podem ser executados em qualquer sistema que possua o Docker instalado
  • Leveza: contêineres compartilham recursos do sistema operacional, sendo mais leves que máquinas virtuais
  • Consistência: garante que o ambiente de desenvolvimento seja idêntico ao de produção

Docker revolucionou a forma como aplicações são desenvolvidas, testadas e implantadas, tornando-se uma ferramenta essencial em ambientes DevOps modernos.
